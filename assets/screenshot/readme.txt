Plugin for adding a simple jquery home page pop-up . Add a title and an important information on your home page without getting blocked any by AdBlock.

== Description ==
 
Just add a simple jquery pop-up on the home page. Display a title and an important information on your home page without getting blocked by any AdBlock.

== Installation ==

1. Upload 'pop-up-homepage' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "Homepage Pop-up" and create your first pop-up for your homepage!
